Happy Friday.

 

Forty-seven percent of CISOs have already observed an AI agent acting without authorization. Only 5% feel confident they could contain one. Those numbers are from the 2026 Saviynt CISO AI Risk Report, and they frame everything in this Meta rogue agent breakdown. The agent wasn't hacked, it held valid credentials. The failure happened after authentication, and nothing in the identity stack caught it.

 

The latest Beyond the Pilot episode covers the other side of that problem. Zoom's AI Companion now tracks opinion divergence in meetings — Sam Witteveen calls it "a land grab for context." More context means more blast radius when something goes wrong.

Around the web

OpenAI is acquiring Astral, the company behind Python's most-used tools.

Astral makes uv (100M+ downloads/month) and ruff, two tools most Python developers now depend on daily. The code is MIT/Apache 2.0 licensed, so forking is possible, but the leadership and product direction are harder to replicate. HN commenters see a pattern: AI companies buying developer tooling infrastructure.

Read more at Astral

 

LlamaIndex ships LiteParse: local document parsing with zero Python dependencies.

A TypeScript-native CLI that parses PDFs, Office docs, and images entirely locally. Uses spatial grid projection instead of trying to convert layouts to markdown. Built-in OCR via Tesseract.js, parallelized across CPU cores. Designed for coding agents that currently write throwaway Python scripts for each new document.

Read more at LlamaIndex

 
 

Xiaomi's MiMo-V2-Pro surpasses 1 trillion parameters, claims to beat Claude Sonnet 4.6 on coding.

42B active parameters, 1M token context. Xiaomi is positioning it as "the brain of agent systems." According to Xiaomi: SWE-bench Verified: 78.0%. ClawEval: 61.5 (No. 3 globally, approaching Claude Opus 4.6's 66.3). Priced at $1/$3 per 1M tokens. Partnerships with OpenClaw, Cline, and Blackbox.

Google Stitch becomes a full AI design canvas with "vibe design" and voice interaction.
